# Sign Flow

Sign Flow is a multitenant SaaS platform designed to manage sales, operations and execution of service projects.

## Core Modules

- Clients
- Leads
- Proposals
- Projects
- Work Orders
- Payments

## Stack

Frontend
- React
- TypeScript
- Vite
- Tailwind
- shadcn/ui

Backend
- Supabase
- PostgreSQL
- RLS

Infrastructure
- Lovable Cloud
- GitHub

## Development

This project is primarily developed in Lovable and automatically synced with GitHub.
